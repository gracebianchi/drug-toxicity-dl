# Drug Toxicity Prediction with Deep Learning

Using deep learning to predict whether chemical compounds are toxic across 12
biological assays. Built in R with the Tox21 dataset (~12,000 compounds) from 
the NIH Tox21 Data Challenge.

Compounds are represented as molecular fingerprints computed from SMILES 
strings and a neural network is trained to classify toxicity across multiple
biological pathways simultaneously.

Built to develop hands-on deep learning experience and explore ML applications
in computational biology and drug discovery.
